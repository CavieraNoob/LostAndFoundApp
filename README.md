# Lost And Found App

## Overview
This is an Android Lost and Found application developed using Java and SQLite database in Android Studio.

The app allows users to:
- Create lost/found adverts
- Upload item images
- Filter items by category
- View advert details
- Remove adverts after the owner is found
- Store data using SQLite database
- Display timestamps for each post

---

## Features

### Create Advert
Users can create a new lost or found advert with:
- Item type (Lost / Found)
- Name
- Phone number
- Description
- Date
- Location
- Category
- Image upload

### View All Items
Users can view all adverts in a list format.

### Filter by Category
Items can be filtered by categories such as:
- Electronics
- Pets
- Wallets
- Keys
- Bags
- Other

### View Details
Users can open an advert to see full details and uploaded image.

### Remove Advert
Users can remove adverts after the item has been returned.

## Technologies Used

- Java

- ListView
- Image Upload

---

## How to Run

1. Open the project in Android Studio
2. Allow Gradle to sync
3. Run the app on an emulator or Android device
